Method 1: Horizontal Stripes (Like Cate's Site)

Hi, I'm Cate.

This is the horizontal stripe background that matches your design!

CSS Code:

background: linear-gradient( to bottom, /* Teal stripe - 0% to 20% of height */ #52B5A8 0%, #52B5A8 20%, /* Pink stripe - 20% to 40% */ #F4B5C4 20%, #F4B5C4 40%, /* Yellow stripe - 40% to 60% */ #F5E6B3 40%, #F5E6B3 60%, /* White stripe - 60% to 80% */ #FFFFFF 60%, #FFFFFF 80%, /* Green stripe - 80% to 100% */ #A8D5BA 80%, #A8D5BA 100% );

Method 2: Repeating Stripes (For Longer Pages)

Repeating Pattern

This pattern repeats automatically, perfect for long scrolling pages.

CSS Code:

background: repeating-linear-gradient( to bottom, #52B5A8 0px, #52B5A8 80px, #F4B5C4 80px, #F4B5C4 160px, #F5E6B3 160px, #F5E6B3 240px, #FFFFFF 240px, #FFFFFF 320px, #A8D5BA 320px, #A8D5BA 400px );

Method 3: Vertical Stripes (Alternative Style)

Vertical Style

Same colors, but vertical orientation. Just change "to bottom" to "to right"!

CSS Code:

background: linear-gradient( to right, /* Changed from "to bottom" */ #52B5A8 0%, #52B5A8 20%, #F4B5C4 20%, #F4B5C4 40%, #F5E6B3 40%, #F5E6B3 60%, #FFFFFF 60%, #FFFFFF 80%, #A8D5BA 80%, #A8D5BA 100% );

Method 4: Diagonal Stripes (Fun Variation)

Diagonal Fun!

45-degree angle stripes for a dynamic look.

CSS Code:

background: repeating-linear-gradient( 45deg, /* Angle instead of direction */ #52B5A8 0px, #52B5A8 40px, #F4B5C4 40px, #F4B5C4 80px, /* ...etc */ );

Method 5: Subtle Stripes (With Opacity)

Subtle Touch

Very light stripes that don't overpower content. Great for backgrounds behind text!

CSS Code:

background-color: white; background-image: repeating-linear-gradient( to bottom, /* Using rgba for transparency */ rgba(82, 181, 168, 0.1) 0px, rgba(82, 181, 168, 0.1) 80px, rgba(244, 181, 196, 0.1) 80px, rgba(244, 181, 196, 0.1) 160px, /* ...etc - last value is opacity (0.1 = 10%) */ );

Real-World Example: Full Page Background

For your entire page:

/* In your global CSS or layout component */ body { background: linear-gradient( to bottom, #52B5A8 0%, #52B5A8 20%, #F4B5C4 20%, #F4B5C4 40%, #F5E6B3 40%, #F5E6B3 60%, #FFFFFF 60%, #FFFFFF 80%, #A8D5BA 80%, #A8D5BA 100% ); background-attachment: fixed; /* Keeps stripes in place when scrolling */ background-size: 100% 100vh; /* Covers full viewport height */ min-height: 100vh; } /* Or for Next.js - in your RootLayout component */ .main-container { background: linear-gradient(...); min-height: 100vh; }

ðŸ’¡ Pro Tips:

  • Percentages vs Pixels: Use percentages (%) for responsive designs, pixels (px) for exact sizes
  • Hard Stops: Use the same color twice (like #52B5A8 0%, #52B5A8 20%) for crisp edges
  • Fixed Attachment: Add background-attachment: fixed to keep stripes stationary while content scrolls
  • Layering: You can layer multiple gradients or put content boxes on top with position: relative
